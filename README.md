# poemfill-site
PoemFill.dev is a sponsor-ready literary prompt builder for generating structured public-domain-inspired prompt instructions. The site uses dynamic frontend controls for theme, authorship, work reference, content type, tone, language style, and composition structure, then assembles a live prompt preview with copy and download support. The current MVP is a static GitHub Pages site with a verified local development workflow through VS Code, GitHub Desktop, and Python’s local HTTP server. Future development will focus on an owner portal for managing prompt controls, improved paragraph/section alignment logic, expanded sponsor placement management, and structured data files for scalable editorial prompt configuration.
