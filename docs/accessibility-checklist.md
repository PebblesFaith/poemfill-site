# Future Accessibility Review Checklist

Review the interface for:

- Keyboard navigation across navigation, form fields, buttons, and concept links
- Visible focus states for interactive elements
- Form label accuracy and field descriptions
- Color contrast across hero, cards, sponsor slots, and form surfaces
- Screen-reader clarity for sponsor placeholders and ad containers
- Proper `aria-label` usage for sponsor containers
- Proper heading order across hero, builder, preview, and portal sections
- Adequate link text for portal, concept, and legal navigation
